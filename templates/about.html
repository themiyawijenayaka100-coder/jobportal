About Job Portal

A simple Django app connecting talent and employers.

Job Portal is a simple web application built using Django that connects job seekers with employers.

Users can register, browse jobs, and showcase their profiles, while employers can post opportunities and find suitable candidates.

This project demonstrates core web development concepts including authentication, role-based access, and CRUD operations.

Features
User registration & login
Edit profile & change password
Job listing & applications
Talent directory
